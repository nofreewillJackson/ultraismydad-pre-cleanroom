A static site is a strong substrate for a garden: every note is pre-rendered to HTML at
build time, so the whole graph is crawlable, fast, and survives without a server. ^thesis

The trade-off is that interactivity has to be layered on with a little client-side
JavaScript — exactly how the [[Wikilinks and Backlinks|link graph]] is drawn. See also
[[Welcome to the Garden]].
